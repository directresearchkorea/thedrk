# Enhancing Game UX with Eye Tracking: 4 Key Applications

Games have evolved beyond simple entertainment into an immersive experience where user experience (UX) and interface design play a pivotal role. Especially in games like mobile PUBG, intricate design and user interfaces (UI) can significantly impact gameplay satisfaction. In this context, Eye Tracking technology has emerged as a powerful tool to analyze player behavior and experience. Let’s explore how eye tracking can be effectively used in game UX research with four specific applications.

4 Key Applications of Eye Tracking in Game UX Research

#### 1. Attention Zone Analysis

Objective: Identify the most focused-on elements on the screen (e.g., UI buttons, maps, characters) and improve areas that lack visibility. Example: If players take too long to locate a particular button, adjusting its size, color, or position can enhance usability and reduce navigation time.

#### 2. Visual Path Analysis

Objective: Track the player’s gaze movements across the screen to evaluate the natural flow of visual information. Example: During combat scenes, if players frequently miss important elements like the health bar, redesigning the HUD to make critical information more prominent can improve gameplay.

#### 3. Fixation and UX Problem Detection

Objective: Measure how long players fixate on certain elements to identify usability issues or difficulty levels. Example: In a puzzle game, if players spend too much time focusing on a specific area, it could indicate overly high difficulty or insufficient hints, prompting adjustments to improve accessibility.

#### 4. Heatmap and Emotional Analysis Integration

Objective: Combine eye-tracking data (heatmaps) with EDA (electrodermal activity) or facial emotion analysis to comprehensively understand players’ emotional responses and UX challenges. Example: If players show frustration or lose interest at specific moments, analyzing their gaze data can reveal which UI or gameplay elements need improvement.

Conclusion: Building Better Games Through the Player's Eyes

Eye tracking in game UX research goes beyond data collection—it uncovers hidden player needs and enables the design of optimized experiences. The examples shared today highlight how this technology can enhance UI/UX design and improve player satisfaction.

By "following the player's gaze," you can create more intuitive, engaging, and enjoyable games. We look forward to seeing eye tracking play a greater role in shaping the future of game design and research.
